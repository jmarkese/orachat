## Ora Chat

To setup:

- [Setup Scotchbox 3.0](https://box.scotch.io/).
- Add `193.168.33.10    orachat.dev` to your Hosts file.
- run `vagrant up --provision`

## Testing

To test:

```
# vagrant ssh
# cd /var/www/
# phpunit
```
